   B. Filename:        traindata_1.csv
      Short description:        The training dataset for Example 1 (800 observations)


        
   C. Filename:        testdata_1.csv
      Short description:	The test dataset for Example 1 (200 observations)

   A. Filename:        1. Generating Data.R
      Short description:        Generate Example data 1 ("data_1.csv"), see supplementary materials for details.

        

   B. Filename:        2. Example 1_LBA_NN.R
      Short description:        Implement LBA-NN on example 1. Can yield (in "Output/"): (1) fig3_importance_plot_example_1.png; (2) fig4_biplot_example_1.png; (3) fig5a_confusion_matrix_lbann.png; (4) tab5_1_summary_prediction_lbann.csv


        
   C. Filename:        3. Example 1_LBA.R
      Short description:	Implement LBA-NN on example 1. Can yield (in "Output/"): (1) tab3_contingency_table_example1.csv; (2) tab4_1_mixing parameters A.csv; (3) tab4_2_latent budgets B.csv; (4) fig5b_confusion_matrix_lba.png; (5) tab5_2_summary_prediction_lba.csv



   D. Filename:        4. function.R
      Short description:	Involved functions. (1) evm.test: generate the input data matrix; (2) rvm.test: generate output data matrix
